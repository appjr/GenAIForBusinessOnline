## How to Use

### For Students:
1. Download the template for your chosen track
2. Rename file to: `LastName_FirstName_Week5_Track#.ipynb`
3. Fill in all TODO sections
4. Complete business analysis questions
5. Test that all code runs
6. Submit with required deliverables

### For Instructors:
- Templates provide consistent structure
- Clear grading rubrics embedded
- TODO markers for easy assessment
- Can be customized per semester

## Template Structure

Each template includes:
1. **Header** - Student info section
2. **Implementation** - Step-by-step coding sections with TODO markers
3. **Analysis** - Business questions and evaluation
4. **Documentation** - Summary and reflection
5. **Checklist** - Pre-submission verification

## Grading Alignment

Templates align with homework assignment document:
- Implementation: 50-60 points
- Business Analysis: 30-35 points
- Documentation: 10-15 points
- **Total: 100 points**
