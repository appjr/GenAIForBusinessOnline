BUAN 6v99.SW2 — Generative AI for Business

🎓 Week 7 Homework

Build a Chatbot for a Real Business Need

📅 Due: Wednesday, March 11 — 3:59 pm 📤 Submit via eLearning

Assignment

Design and build a working AI chatbot that solves a real business problem. Your chatbot must use the OpenAI or Anthropic API, include a RAG knowledge base, and be deployed as a Gradio web app.

Deliverables

  1. Working Chatbot — a Python implementation with a clear business use case and an appropriate persona defined via a system prompt.
  2. System Prompt Documentation — your system prompt with a short explanation of the design decisions you made.
  3. Knowledge Base — at least 10 documents or text chunks loaded into a vector store for RAG.
  4. Gradio Deployment — a shareable Hugging Face Spaces URL, or a screenshot/screen recording if a public URL is not available.
  5. Written Writeup (1–2 pages) answering:
    • What problem does the chatbot solve?
    • Who is the target user?
    • What guardrails did you implement?
    • How would you measure success in production?
    • What would you do differently with more time?

Grading

Component Weight
Working chatbot with appropriate persona 30%
RAG integration with relevant knowledge base 25%
Guardrails and safety considerations 20%
Gradio deployment 10%
Business case writeup 15%
Total 100%
⭐ Bonus

Integrate function calling to connect your chatbot to a real external API (weather, stock prices, news, etc.)

📦

Submission Instructions

Submit a zip file containing your Python code + writeup PDF via eLearning by Wednesday, March 11 at 3:59 pm. Late submissions will not be graded.
